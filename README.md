# Interactive Quiz Application

A premium interactive quiz application built using vanilla HTML, CSS, and JavaScript.

## Features
- Category-based quizzes (Cricket, GK, AP Politics, Science, Technology)
- Randomized questions on every reset
- Instant feedback with scoring
- Clean, modern, responsive UI
- No external libraries or frameworks

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Clone the repository
2. Open index.html in your browser

## Future Enhancements
- Timer-based questions
- Difficulty levels
- Leaderboard support

Built as part of an interactive frontend project.
